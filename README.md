# Wrist-Relief AI (Digital Mockup - HCI)

A modern web application designed to help users monitor and manage wrist health through AI-powered insights and tracking.

## Overview

Wrist-Relief AI is an interactive web application that provides users with tools to track wrist discomfort, receive AI-powered recommendations, and maintain better wrist health through data visualization and personalized insights.

## Features

- **Interactive Mobile-First Design**: Responsive interface optimized for mobile devices with a phone frame design for desktop viewing
- **AI-Powered Recommendations**: Get personalized advice and insights for wrist health management
- **Data Visualization**: Track wrist health metrics with Chart.js integration
- **Markdown Support**: Render rich content using marked.js
- **Modern UI**: Built with Tailwind CSS featuring neon-themed cyberpunk aesthetics
- **Project Portfolio**: Comprehensive project documentation and showcase page

## Files

- `index.html` - Main application interface
- `portfolio.html` - Project portfolio and documentation page

## Technologies Used

- **HTML5** - Structure and markup
- **Tailwind CSS** - Utility-first CSS framework
- **Chart.js** - Data visualization and charting
- **Marked.js** - Markdown parsing and rendering
- **Google Fonts** - Rajdhani and Inter font families

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser to launch the application
3. View `portfolio.html` for project documentation and details

## Design

The application features a modern, cyberpunk-inspired design with:
- Dark slate background colors
- Neon cyan and purple accents
- Custom scrollbar styling
- Grid pattern backgrounds
- Phone frame interface for desktop viewing
- Responsive design for mobile and desktop devices

## Browser Compatibility

This application works best in modern browsers that support:
- ES6+ JavaScript
- CSS Grid and Flexbox
- CSS Custom Properties
